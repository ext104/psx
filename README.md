### PSX - PS[1-2]

Installation:
```
source psx/install.sh
```
or
```
echo -e "alias psx='source $HOME/psx/psx'\n\
psx run" >> ~/.bashrc
```
Project hierarchy:
```
psx/ - Project directory
 - profile.d/  - Directory with profiles
 - template.d/ - Directory with templates for profiles
 - psx      - Utility
 - psx.conf - Project variables 
```
